### EXECUTION PLAN

The `exec-plan.md` file contains the current execution progress table. Read it at the start of each session to understand what has been completed and what is pending.

After completing your task:
- The DAG engine will automatically update `exec-plan.md` at the start of the next session
- You do NOT need to manually update `exec-plan.md`
- Update `task_list.json` fields instead: `passes`, `rework`, `rework_note`, `rework_count`
