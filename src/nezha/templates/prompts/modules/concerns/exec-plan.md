### EXECUTION PLAN

The `{{workspace}}/exec-plan.md` file contains the current execution progress table. Read it at the start of each session to understand what has been completed and what is pending.

After completing your task:
- The DAG engine will automatically update `{{workspace}}/exec-plan.md` at the start of the next session
- You do NOT need to manually update `{{workspace}}/exec-plan.md`
- Update `{{workspace}}/task_list.json` fields instead: `passes`, `rework`, `rework_note`, `rework_count`
